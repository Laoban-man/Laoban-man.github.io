Cell classification in leukemia

Image classification

Cell classification in leukemia remains a challenge today due to the similarities between healthy and sick cells. Human error is widespread and the task is time-consuming.

This project aims to develop a tool to help with cell classification using progressively more complex machine learning algorithms and then provides a tool to explore and deploy the solution.
